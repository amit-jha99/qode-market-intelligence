
# Market Intelligence & Sentiment Analysis Pipeline

## Overview
This project implements an end-to-end market intelligence pipeline that collects market-related tweets, processes and deduplicates the text data, stores it efficiently, and generates a market sentiment signal with a confidence estimate.

The primary objective of this project is to demonstrate data collection, processing, storage, and analysis workflows following professional software development practices.


---

## Setup Instructions

### 1. Clone the Repository

git clone <repository-url>
cd qode-market-intelligence

### 2.Install Dependencies
pip install -r requirements.txt


### How to Run
python main.py
